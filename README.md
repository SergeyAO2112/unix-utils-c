# unix-utils-c
# Unix Utilities in C

Implementation of basic Unix utilities in C.

## Implemented tools

### cat

- supports flags: -b -n -s -e -t -v
- processes input from files and stdin

### grep

- supports flags: -e -i -v -c -l -n
- pattern matching in text files

## Features

- command-line argument parsing
- file I/O handling
- error handling
- modular project structure

## Technologies

- C
- Linux
- Makefile
